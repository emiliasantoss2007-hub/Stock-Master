# FORMATAÇÃO (ABNT NBR 14274: 2011)
- **Formato do pape**l: A4 branco ou reciclado;
- **Fonte**: Times New Roman/tamanho 12 para todo o trabalho, “inclusive capa,
excetuando-se citações com mais de três linhas, notas de rodapé, paginação, [...]
legendas e fontes das ilustrações e das tabelas, que devem ser em tamanho menor e
uniforme (tamanho 10);
- **Espaçamento**: 1,5, sendo o texto disposto em uma coluna, “excetuando-se as
citações de mais de três linhas, notas de rodapé, referências, legendas das ilustrações
e das tabelas, natureza”. Para artigos e relato tecnológico recomenda-se
espaçamento simples, conforme a NBR 6022: 2018 e a NBR 10719: 2015.
- **Alinhamento**: justificado
- **Margens**: para anverso, superior: 3 cm; inferior: 2 cm; esquerda: 3 cm; direita: 2 cm;
para verso, superior 3 cm; inferior 2 cm, esquerda 2 cm; direita 3 cm.
- **Parágrafo**: 1 tab
- **Citações e notas**: devem ser conforme a ABNT NBR 10520. Notas de tabelas devem
ser conforme as Normas de apresentação tabular do Instituto Brasileiro de Geografia
e Estatística (IBGE)
- **Seções**: os títulos das seções com ou sem indicativo numérico devem ser conforme
a ABNT NBR 6024: 2012.
- **Impressão**: recomenda-se que os elementos textuais e pós-textuais sejam digitados
no anverso e verso das folhas

# CONFIGURAÇÃO GERAL DA PÁGINA

- [ ] Papel A4 (21 cm × 29,7 cm)

- [ ] Margem esquerda: 3 cm
- [ ] Margem inferior: 2 cm
- [ ] Margem direita: 2 cm



# FONTE

- [ ] Fonte Arial ou Times New Roman
- [ ] Tamanho 12 no texto principal
- [ ] Tamanho 10 para:
  - citações longas;
  - notas de rodapé;
  - paginação;
  - legendas;
  - ficha catalográfica.



# ESPAÇAMENTO

- [ ] Espaçamento 1,5 no texto principal
- [ ] Espaçamento simples em:
  - citações longas;
  - referências;
  - legendas;
  - notas de rodapé;
  - ficha catalográfica.



# PARÁGRAFOS

- [ ] Recuo da primeira linha: 1,25 cm
- [ ] Texto justificado
- [ ] Sem espaço antes/depois do parágrafo



# PAGINAÇÃO

- [ ] Numeração iniciada na folha de rosto
- [ ] Numeração visível apenas a partir da introdução
- [ ] Número da página no canto superior direito

## Seções secundárias (Título 2)

* [ ] Primeira letra maiúscula
* [ ] Negrito

## Seções terciárias (Título 3)

* [ ] Primeira letra maiúscula


# CITAÇÕES — ABNT NBR 10520

## Citação curta (até 3 linhas)

* [ ] Dentro do parágrafo
* [ ] Entre aspas
* [ ] Autor + ano + página

### Exemplo:

```text
Segundo Silva (2020, p. 15), "a rastreabilidade é fundamental".
```



## Citação longa (mais de 3 linhas)

* [ ] Recuo de 4 cm
* [ ] Fonte 10
* [ ] Espaçamento simples
* [ ] Sem aspas
